# mytens-assessment
MyTens Assessment, Telkom Digital Solution by PT Telekomunikasi Indonesia

to start project:
- clone repo,
- open to Editor
- yarn install/npm install
- yarn start/npm start
- yarn android/npm run android (emulator/real device is ready)
